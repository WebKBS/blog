## 1. React Swiper 라이브러리 설치

먼저 React Swiper 라이브러리를 설치합니다.

<MDXLink href="https://swiperjs.com/get-started" title="Swiper 공식홈페이지 " />

```bash
npm install swiper
```

swiper를 설치하셨다면 이제 swiper css를 위치에 맞게 import 해줍니다.
